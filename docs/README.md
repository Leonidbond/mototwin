# MotoTwin Docs Index

Папка `docs/` содержит техническую и продуктовую документацию по текущему MotoTwin MVP: архитектура, модель данных, API, frontend-реализация, функциональная логика и правила работы в проекте.

## Документы

- `technical-overview.md` — общий технический обзор текущей реализованной архитектуры и модулей MVP.
- `data-model.md` — описание Prisma-модели данных: сущности, связи, enum и текущие ограничения.
- `api-backend.md` — документация по реализованным backend routes, валидациям и бизнес-правилам.
- `frontend.md` — обзор страниц и UI-потоков текущего frontend.
- `functional-logic.md` — сквозная логика обслуживания: статусы узлов, правила, агрегация и пересчет.
- `coding-rules.md` — правила разработки и код-стандарты проекта.
- `cursor-workflow.md` — правила и практики работы с Cursor в проекте.
