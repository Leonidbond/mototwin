## Полное описание

```
MotoTwin — цифровой гараж для владельца мотоцикла. В одном приложении — профиль техники, журнал обслуживания, напоминания по ТО, учёт расходов и подбор совместимых деталей по узлам.

Для кого
• Хотите видеть полную сервисную историю, а не разрозненные заметки и чеки
• Нужен контроль пробега, состояния узлов и затрат на владение

Что умеет приложение

Гараж и профиль мотоцикла
Добавьте мотоцикл: марка, модель, модификация, пробег и параметры эксплуатации. Всё в одной структурированной карточке.

Журнал обслуживания
Фиксируйте ТО, замену расходников и ремонты: дата, пробег, узел, работы, детали и стоимость. История всегда под рукой — при продаже, на сервисе или в гараже.

Статусы и напоминания
Сразу видно, какие узлы в порядке, что скоро потребует внимания и что просрочено. Push-уведомления напомнят о важном (подключаются в настройках).

Расходы
Учитывайте детали, работы и расходники — понимайте реальную стоимость владения, а не только цену бензина.

Подбор деталей
Ищите совместимые запчасти по узлу мотоцикла на основе каталога и fitment-логики, а не по догадкам в чатах.

Вход и синхронизация
Войдите по email или через Google / Yandex. Данные хранятся в облаке и доступны с телефона и в браузере на mototwin.space.

MotoTwin развивается: сейчас поддерживаются популярные модели (в т.ч. BMW и KTM), каталог и функции расширяются.

Политика конфиденциальности: https://mototwin.space/privacy
Поддержка: support@mototwin.online
```

## Пошаговое заполнение Play Console

Откройте [play.google.com/console](https://play.google.com/console) → приложение **MotoTwin** (`ru.mototwin.app`).

### 1. Страница приложения

**Путь в меню (русский интерфейс Play Console):**

1. [play.google.com/console](https://play.google.com/console)
2. **Все приложения** → выбрать **MotoTwin**
3. Слева: **Привлечение пользователей**
4. Подменю: **Страница приложения**
5. Пункт: **Основная страница приложения в Google Play**

На этой странице два блока:

| Блок на странице | Что заполнить |
|------------------|---------------|
| **О приложении** | Название, краткое и полное описание |
| **Графика** | Иконка, промо-графика (feature graphic), скриншоты телефона |

| Поле | Значение |
|------|----------|
| Язык по умолчанию | Русский (если спросят при создании — или **Управление переводами**) |
| Название приложения | `MotoTwin — гараж мотоцикла` |
| Краткое описание | `Журнал ТО, напоминания, расходы и подбор деталей для вашего мотоцикла` |
| Полное описание | см. блок «Полное описание» выше в этом файле |
| Иконка приложения | `images/play-store/icon-512.png` |
| Промо-графика | `images/play-store/feature-graphic-1024x500.png` |
| Скриншоты телефона | 6–8 файлов из `images/play-store/` |

**Категория и контакты** — отдельная страница в том же подменю:

6. **Привлечение пользователей** → **Страница приложения** → **Настройки для Google Play**

| Раздел на странице | Значение |
|--------------------|----------|

### 4. Рейтинг контента (Content rating)

**Политика приложения → Рейтинг контента** → начать анкету IARC.

| Вопрос | Ответ |
|--------|-------|
| Email для сертификата | `support@mototwin.online` |
| Категория | Utility, Productivity или Reference |
| Насилие, секс, наркотики, азартные игры | Нет |
| Пользовательский контент | Пользователь вводит данные о своём мото (не публичная соцсеть) |
| Обмен сообщениями между пользователями | Нет |
| Покупки | Нет in-app purchases |
| Расположение / контакты | Не собираем для функций приложения |

Ожидаемый рейтинг: **3+ / Everyone** (утилита без чувствительного контента).

### 5. Целевая аудитория (Target audience)

**Политика приложения → Целевая аудитория**:

- Приложение **не** ориентировано на детей.
- Возраст: **13+** или **16+** (владельцы мотоциклов).
- **Не** участвовать в программе «Designed for Families».

### 6. Безопасность данных (Data safety)

**Политика приложения → Безопасность данных**:

**Общие:**

| Вопрос | Ответ |
|--------|-------|
| Собираете или передаёте данные? | Да, собираем |
| Все данные шифруются при передаче? | Да (HTTPS) |
| Можно запросить удаление данных? | Да → `support@mototwin.online` |
| Данные продаются? | Нет |
| Используются для рекламы? | Нет |

**Типы данных (Collected, not shared, optional/required as noted):**

| Тип в форме Play | Собираем | Обязательно | Цель |
|------------------|----------|-------------|------|
| Email address | Да | Да | Account management |
| Name | Да | Нет (OAuth) | App functionality |
| Other user-generated content (мото, ТО, пробег) | Да | Нет | App functionality |
| Device or other IDs (push token) | Да | Нет | App functionality |

**Передача третьим лицам:** только процессоры (OAuth Google/Yandex, FCM для push, хостинг) — не «sharing» в смысле продажи; в форме отметить, что данные **не** передаются для рекламы и **не** продаются.
